Como usar:

  • Puedes agregar "jugadores" con el boton "Agregar jugador, se desplegar√° una vista para seleccionar sus horas libres
  • Selecciona las horas libres haciendoles click, estas se colocaran en verde
  • Puedes mantener click y arrastrar el mouse sobre las horas para seleccionar varios a la vez
  • Puedes mantener click y arrastrar el mouse sobre las horas ya seleccionadas para deseleccionar las que ya seleccionaste
  • Puedes hacer click en el nombre del dia de la semana para seleccionar o deseleccionar todas las horas
  • Cuando hallas terminado escribe el nombre de el jugador en la parte superior
  • Cuando hallas terminado de agregar a todos los jugadores puedes darle al boton "Calcular" para obtener un horario donde todos los jugadores estan libres

Este proyecto lo hice en 2 dias, 2 horas por dia en javascript vanilla e imitando un patron similar al de react, lo hice con vanilla por querer probar mis propios conocimientos sobre javascript web vanilla pero en si, para aplicaciones de esta indole con este tipo de logica compleja es mejor utilizar algun framework como seria react o angular.
